# Pookal

Pookal is a florist commerce platform starter built around the architecture we mapped for a modern flower retail business:

- React + Vite + Bootstrap frontend
- Laravel-style backend structure
- MySQL-ready configuration and migrations
- Modules for POS, inventory, orders, CRM, delivery, reporting, and settings

## Workspace Structure

```text
pookal/
  frontend/   # React + Vite + Bootstrap client
  backend/    # Laravel-style API skeleton
```

## Frontend Modules

- Dashboard
- POS
- Inventory
- Orders
- CRM
- Delivery
- Reports
- Settings

## Backend Domains

- Authentication placeholder
- Product catalog
- Inventory
- Orders
- Customers
- Delivery
- Reporting

## Suggested Next Steps

1. Run `npm install` inside `frontend`
2. Install Laravel dependencies inside `backend` if you want a fully runnable app
3. Wire frontend services to the backend API routes
4. Add authentication, validations, and database seeders
